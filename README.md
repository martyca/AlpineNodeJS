# AlpineNodeJS
docker build of NodeJS for Alpine

## Versions
Alpine 3.11.6
NodeJS 12.15.0
NPM    6.13.4

## References
### Alpine
https://hub.docker.com/_/alpine
### NodeJS
https://nodejs.org
### NPM
https://www.npmjs.com

## Instructions:
### Pull
`docker pull martyca/alpinenodejs`
### Run
`Docker run -it martyca/alpinenodejs sh`
